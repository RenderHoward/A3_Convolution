Prerequisites:

MATLAB or Octave
    
Files:

> Input data: 

>> `convolve_raw_0.dat`, `convolve_raw_1.dat`

> Code:	
>> `Convo.m`

> Output plots:
>> `raw0.pdf`, `raw1.pdf`, `ConvolutionResult.pdf`


`Convo.m` loads given input data files, performs convolution on them and plots resulting function over enlarged domain.
